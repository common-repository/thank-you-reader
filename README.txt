Simple plugin to appreciate the reader or to say thank you.

== Description ==

A simple lightweight plugin which will automatically print thank you message at the end of post. No setup is required. Install and it will automatically start working. Just write the post, publish it and forget about camelcase, plugin will automatically make the changes.

== Installation ==

Just upload & activate the plugin and ready to work.
